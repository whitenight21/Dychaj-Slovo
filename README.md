# Dýchaj Slovo – Breathe the Word

Welcome to **Dýchaj Slovo** – a simple, peaceful spiritual platform focused on breathing in God's Word through daily inspiration, short reflections, and family-friendly Christian resources.

Live website:  
🌐 https://whitenight21.github.io/Dychaj-Slovo/

Language: **Slovak** (with gradual English expansion planned)

## What is Dýchaj Slovo?

"Dýchaj Slovo" means **"Breathe the Word"** in Slovak.  
The project helps people slow down, breathe deeply, and let Scripture nourish their hearts and families — one breath, one verse, one small step at a time.

## Current Features (March 2026)

- Clean, mobile-friendly single-page design
- Product showcase with spiritual e-books and printable worksheets:
  - Personal Spiritual rtreats
  - Reflection materials
  - Coloring books and creative activities for children
  - Family devotion resources
- Direct links to view publications on Amazon
- Shopping cart functionality (`kosik.html`, `potvrdenie.html`, `doprava.html`, etc.)
- High-quality cover images and product previews
- Order information and confirmation pages (`informacie.html`, `zhrnutie.html`)
- GitHub Pages hosting (free, fast, always available)

## Planned Features & Roadmap

### Short-term (next 1–3 months)

- English translation of main content and product descriptions
- Better responsive layout for tablets and larger screens
- Real "Buy now" buttons linking to Amazon product pages
- Simple contact form or email link for questions / custom orders
- Separate pages for each major product series (Adam, Advent, Kids coloring, etc.)
- Dark mode toggle
- Improved image loading (lazy loading, WebP versions)

### Medium-term (3–9 months)

- Blog / daily short devotion section (1–2 verses + reflection)
- Printable PDF download buttons for free worksheets (direct from site)
- Search function across products and reflections
- Multi-language switcher (Slovak ↔ English at minimum)
- Integration with GitHub Actions for automatic deployment previews

### Long-term ideas (9+ months)

- User accounts / wish-list (requires backend → consider migrating to Next.js or similar)
- Newsletter signup (Mailchimp / Buttondown integration)
- Audio versions of short reflections (recordings)
- Mobile app version (PWA – Progressive Web App)
- Community section (testimonials, shared experiences)
